## Wiki Maintenance

### Regular Updates Needed

- Add new commands to Commands-Overview.md
- Update agent pages if capabilities change

**Monthly:**

**Quarterly:**
- Full content review for accuracy
- Update screenshots if UI changed
- Review SEO and discoverability
- Check for dead links

### Content Guidelines

**Writing Style:**
- Clear, concise, professional
- Active voice
- Present tense
- Second person ("you")

**Code Examples:**
- Always include language identifier in code blocks
- Use realistic examples
- Include comments for complex code
- Show both good and bad examples where appropriate

**Links:**
- Use relative links between wiki pages: `[Installation](Installation)`
- Use full URLs for external links
- Always provide link text, never raw URLs
- Check links regularly

**Formatting:**
- Use H2 (##) for main sections
- Use H3 (###) for subsections
- Use tables for comparisons
- Use lists for steps and items
- Use code blocks for commands and code
- Use blockquotes for important notes
